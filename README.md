- 오픈 디시콘 포멧으로 작동하는 코드가 아닌, 명령어당 하나의 링크를 등록시켜 보내주는 굉장히 수준이 낮은 코드입니다.

- 빗금 명령어를 통한 디시콘/이미지 호출
